Chapter 01

超音波の基礎物理

POCUSを理解するための物理的基盤。波の振る舞い、エネルギーの伝搬、そして画像形成の仕組みを体系的に学びます。

📚 D4.1 〜 D4.11 🎯 11項目 ⏱ 推定読了 25分
波の基本パラメータ
D4.1
周波数
Frequency
▾
🔰初学者向け説明
ラジオ局の「周波数」をイメージしてください。周波数が高い（AM/FMの数字が大きい）ほど、信号が細かく振動し、情報を精細に伝えられますが、壁などの障害物で遮断されやすくなります。超音波も同じで、高周波数＝精細だが遠くに届かない、低周波数＝粗いが遠くまで届くという関係があります。
📖定義
波の振動（周期）が1秒間に何回繰り返されるかを示す数値。単位は Hz（ヘルツ）。人間の可聴域は約20Hz〜20,000Hz（20kHz）であり、それを超える音波が超音波。診断用超音波は 2〜20 MHz の範囲が用いられる。
⚙️メカニズム
  • 音速(c) = 周波数(f) × 波長(λ) ── 媒質中の音速は一定なので、fが上がるとλは短くなる（反比例）
  • プローブの振動子は厚み(L)で決まる共振周波数 fz = c / 2L を持つ
  • パルス波は単一周波数でなく周波数スペクトルを持ち、頂点を中心周波数、半値幅を帯域幅と呼ぶ
  • 高周波数ほどビームは絞られ（指向性↑）、減衰も大きくなる
🏥臨床的意義
  • 高周波（7.5〜15MHz）：頸動脈・甲状腺・表在リンパ節など浅部。分解能◎ 深達性✕
  • 低周波（2〜5MHz）：腹部・心臓など深部。分解能△ 深達性◎
  • ドプラシフト（血流速度の計測）は送信周波数に依存 → 低周波プローブはシフト量が小さく、エイリアシングが起きにくい
  • 高周波・高出力は熱発生（TI上昇）に関与 → ALARA原則を守る
🔍関連概念・鑑別
  • 周波数 vs PRF：前者は超音波の振動数、後者は1秒間の送受信回数（パルス繰り返し）。別概念。
  • 中心周波数 vs 帯域幅：中心周波数はプローブの「得意な音程」、帯域幅は許容する音程の広がり。広い帯域幅は距離分解能向上に寄与。
⚠️注意点
「周波数を上げれば画像がきれいになる」は正しいが、深部が見えなくなるトレードオフがある。腹部エコーでリニアプローブを使うと肝深部が減衰で真っ黒になるのはこのため。
D4.2
波長
Wavelength
▾
🔰初学者向け説明
水面の波紋で「山と山の間の距離」を想像してください。それが波長です。細かい波紋（高周波）は「山と山」が近いので波長が短く、ゆったりした波（低周波）は波長が長くなります。波長が短いほど、細かいものを見分ける力（分解能）が高いため、超音波では周波数を上げて波長を短くすることが鮮明な画像の鍵になります。
📖定義
波の「密」から次の「密」（または「山」から次の「山」）までの距離。記号 λ（ラムダ）、単位 m（メートル）。
c = f × λ より、λ = c / f。5MHz水中では約0.3mm。
⚙️メカニズム
  • 媒質中の音速は一定 → 周波数と波長は反比例（f↑ = λ↓）
  • 距離分解能の理論的限界は約λ/2（波長の半分以下の物体は分離できない）
  • グレーティングローブ：振動子間隔(d)が波長の整数倍の方向に不要なビームが出現。高周波（短波長）ではエレメントピッチを小さくする必要がある
  • 振動子口径が波長に対して十分大きくない場合、球面状に拡散する
🏥臨床的意義
波長は直接操作できないが、周波数を選択することで間接的に制御する。距離分解能を上げたい場合 → 高周波プローブ（短波長）を選択。
🔍関連概念・鑑別
波長 vs 周期(T)：波長は「空間的な長さ」、周期は「時間的な長さ（1振動にかかる秒数）」。T = 1/f。
⚠️注意点
生体内（軟部組織）の音速は約1,540 m/s（装置の計算に使われる固定値）。実際には組織ごとにわずかに異なるため、この差がアーチファクトの原因になることがある。
D4.3
振幅
Amplitude
▾
🔰初学者向け説明
スピーカーのボリューム（音量）をイメージしてください。ボリュームが大きい＝振幅が大きい＝音が強い、です。超音波では振幅が大きいほど強い音波を送り込んでいることになり、画面上のエコーの明るさ（輝度）は振幅の大きさで決まります。ただし体の奥へ進むほど振幅は弱まり（減衰）、深部のエコーは暗くなります。
📖定義
波の振れ幅。音波の強弱の幅であり、超音波の「強さ（Power）」を表すパラメータ。Aモードでは縦軸、Bモードでは輝度（明るさ）として表示される。
⚙️メカニズム
  • 生体内での振幅は伝搬距離に対して対数的（指数関数的）に減衰
  • 例：5MHzで8cm伝搬すると振幅は約1/10になる
  • これを補正するのが STC / TGC（深さに応じて受信ゲインを増幅する機能）
  • 振幅（音圧）は機械的指数 MI（Mechanical Index） に関与
🏥臨床的意義
  • Aモード：振幅を縦軸グラフで表示（眼科など）
  • Bモード：振幅→輝度変換で断面像を形成（現在の主流）
  • TGCを適切に調整することで深部の均一な画質を確保
  • 高出力（大振幅）はキャビテーションリスク → ALARA原則
🔍関連概念・鑑別
振幅 vs 周波数：振幅は「音の強さ（量）」、周波数は「音の高さ（質）」。どちらも独立して設定できるパラメータ。
⚠️注意点
ゲイン（増幅）を上げすぎると、弱いエコー同士の差が潰れて分解能が実質的に低下する。「明るければよい」ではなく、適切な出力設定が正確な診断の前提。
伝搬現象
D4.4
減衰
Attenuation
▾
🔰初学者向け説明
遠くにいる人の声が小さくなるように、超音波も体の奥へ進むほど弱まります。さらに周波数が高い（音程が高い）ほど、この弱まり方が激しくなります。深部が画像上で暗くなる主な原因はこの減衰で、装置のTGCでそれを補正します。
📖定義
超音波が媒質を伝搬する際に、振幅（強度）が距離とともに減少する現象。構成要素は①吸収 ②拡散 ③散乱の3つ。単位：dB/cm（または dB/cm/MHz）。
⚙️メカニズム
  • 振幅は伝搬距離に対して対数的に減衰（dBスケールで線形）
  • 周波数依存減衰：高周波ほど減衰係数が大きい（軟部組織では約0.5 dB/cm/MHz）
  • 5MHz・8cmで振幅が1/10になる例がある
  • 補正手段：STC（Sensitivity Time Control）/ TGC（Time Gain Compensation）
🏥臨床的意義
  • 音響陰影：骨・結石など強反射体の後方にエコーが届かず暗くなる → 結石や石灰化の診断サイン
  • 後方エコー減衰：腫瘤内で減衰が強い場合（例：線維腺腫）に後方が暗くなる
  • 脂肪肝：散乱・吸収増大 → 深部急激に減衰 → 「ブライトリバー」＋深部暗化
🔍関連概念・鑑別
  • 音響陰影 vs 後方エコー減衰：前者は骨・結石など強反射で後方が完全欠損、後者は腫瘤などで相対的に暗い
  • 後方エコー増強：嚢胞など液体（減衰小）の後方では逆に明るくなる
⚠️注意点
TGCを過剰補正すると深部が明るくなりすぎ、病変を見落とす可能性がある。「均一な明るさ」を意識して調整する。
D4.5
吸収
Absorption
▾
🔰初学者向け説明
電子レンジが食品を加熱する仕組みと似ています。超音波の振動エネルギーが組織に吸収され、摩擦熱として変換されます。これが減衰の主な原因であり、安全性（TI：熱指数）の観点から重要です。
📖定義
超音波の音響エネルギーが生体組織を通過する際に熱エネルギーへ変換されて消失する現象。減衰の構成要素（吸収・拡散・散乱）のうち最も主要な要因。
⚙️メカニズム
  • 振動により組織分子間で摩擦が生じ、熱に変換される
  • 高周波数ほど振動回数が多い → 吸収大 → 発熱大
  • 安全指標：TI（Thermal Index） = 組織の温度上昇を推定する指数
🏥臨床的意義
胎児検査など感受性の高い組織では TI を低く保つことが推奨される。TI < 0.7 を目安に管理し、1を超えた場合は照射時間を最小化する。
🔍関連概念・鑑別
吸収 vs 散乱 vs 拡散：3つ合わせて「減衰」。吸収はエネルギーが熱に変わる、散乱は無数の方向に飛ぶ、拡散はビームが広がることによる強度低下。
⚠️注意点
ALARA原則（As Low As Reasonably Achievable）：診断目的を達成できる最小の出力・最短の時間にとどめる。特に妊娠初期・眼球・新生児では厳守。
D4.6
屈折
Refraction
▾
🔰初学者向け説明
水が入ったコップにストローを入れると、水面で「折れ曲がって」見えます。光が空気から水に入るときに屈折するのと同様に、超音波も音速の異なる組織の境界で斜めに当たると進行方向が曲がります。この現象が一つの血管が二重に見える（虚像）などのアーチファクトを引き起こすことがあります。
📖定義
音速が異なる媒質の境界に超音波が斜めに入射した際、進行方向が曲げられる現象。境界に垂直（直角）に入射した場合は屈折しない。スネル（Snell）の法則に従う。
⚙️メカニズム
  • Snellの法則：sin θ₁ / c₁ = sin θ₂ / c₂（入射角と屈折角は音速の比で決まる）
  • 波面の片側が先に速い媒質に入ることで進路が曲がる
  • 応用：音響レンズはこの原理を利用してビームをスライス方向に収束させる
🏥臨床的意義
  • 腹直筋で屈折 → 上腸間膜動脈などが二重に見える（ダブルイメージ）
  • 対処法：プローブの位置・角度を変えて多方向から確認する
  • 音響レンズの設計に屈折を積極的に活用（厚み方向の分解能向上）
🔍関連概念・鑑別
屈折 vs ミラーイメージ：屈折は音波が曲がることで虚像が生じる、ミラーイメージは強い反射面（横隔膜など）に実像が映り込む現象。発生メカニズムが異なる。
⚠️注意点
スライス幅アーチファクトの原因は屈折ではなく、ビームの厚み（スライス厚）の問題。混同しやすいため注意。
D4.7
散乱
Scattering
▾
🔰初学者向け説明
霧の中でヘッドライトを当てると、光が霧の水滴に当たって四方八方に散らばります。超音波でも、波長より小さな微細構造（細胞など）に当たると、無数の反射波が生まれてあちこちに飛びます。これがエコー画像の「粒々感（スペックル）」の正体です。
📖定義
波長より小さい不規則な構造（組織の不均一性など）に超音波が当たることで、あらゆる方向に無数の反射波が生じる現象。これらの散乱波が干渉し合い、画像上にスペックル（speckle）を形成する。
⚙️メカニズム
  • 散乱体（赤血球・組織実質の微細構造）からの反射波が互いに干渉 → スペックルパターン形成
  • スペックルパターンは周波数に依存：高周波ほど細かいパターン
  • 散乱は減衰（吸収+拡散+散乱）の一因でもある
🏥臨床的意義
  • 脂肪肝：脂肪滴による散乱↑ → 肝実質が高輝度（ブライトリバー）
  • 線維化：線維組織の増加 → 散乱パターン変化 → 診断に応用
  • ドプラ法での血流計測：赤血球からの散乱波をドプラシフトとして検出
🔍関連概念・鑑別
散乱 vs 鏡面反射：散乱は小さな不規則構造で「全方向に」飛ぶ、鏡面反射は平滑な大きな境界面で「特定方向に」返る。臓器の輪郭（鏡面反射）とその内部（散乱）を使い分けている。
⚠️注意点
スペックルは組織の実際の構造を反映しているわけではなく、散乱波の干渉パターン。「粒々」が見えても、それが個々の細胞に対応しているわけではない。
境界での現象と画像形成
D4.8
反射
Reflection
▾
🔰初学者向け説明
山びこ（エコー）と全く同じ原理です。声を出すと山に当たって返ってくるように、超音波を体に送ると組織の境界で跳ね返ってきます。この「戻ってきた音（エコー）の強さと時間」を使って画像が作られます。空気や骨は非常に強く反射するため、その先には超音波が届かず、画面が真っ暗になります。
📖定義
音響インピーダンス（Z）が異なる媒質の境界で超音波が跳ね返る現象。
Z（音響インピーダンス）= ρ（密度）× c（音速）
反射率(R) = (Z₂ - Z₁) / (Z₂ + Z₁)
差が大きいほど反射が強く、透過が少なくなる。
⚙️メカニズム
  • 肝臓と筋肉の境界：Z差が小さい → 弱い反射（R ≒ 0.01 → 1%反射）
  • 生体と空気の境界：Z差が極大 → ほぼ100%反射（R ≒ -1.00）
  • ビームが境界面に垂直（90°）に当たるほど反射が強い → より白く映る
  • 鏡面反射（平滑面）vs 散乱（粗面・微細構造）
🏥臨床的意義
  • Bモード画像：エコー強度を輝度に変換して断面像を形成（白＝強反射、黒＝無反射）
  • ゼリー（ジェル）の役割：プローブ-皮膚間の空気（全反射）を排除し透過性向上
  • アーチファクト：多重反射（リバーブレーション）・ミラーイメージ・音響陰影
🔍関連概念・鑑別
  • 反射 vs 透過：一方が増えれば他方が減る（保存則）。両者の和 = 100%。
  • 音響インピーダンス vs 音速：Zは密度も含めた複合指標。音速が同じでも密度が違えばZが異なる。
⚠️注意点
腸管ガスや肺（空気）は超音波の大敵。エコーゼリーを十分に塗り、プローブを体壁に密着させることで空気層をなくすことが重要。
D4.9
透過
Transmission
▾
🔰初学者向け説明
サングラスのレンズは光の一部を通し（透過）、一部は吸収・反射します。超音波も同様に、組織の境界で「反射する量」と「通り抜ける量（透過）」が決まります。透過しやすいほど深部まで届き、液体（嚢胞など）は透過性が高いため、後方のエコーが逆に明るくなります（後方エコー増強）。
📖定義
超音波が組織境界を通り抜けてより深部へ進むこと。反射率と透過率の和は常に1（100%）。音響インピーダンスの差が小さいほど透過率が高い。
⚙️メカニズム
  • Z差が小→透過多（軟部組織間：99%が透過）
  • Z差が大→透過少（骨・ガス：ほぼ透過しない）
  • 周波数が高いほど透過性は低下（深達性↓）
  • 整合層（matching layer）：振動子-生体間のZ差を埋め、効率的に超音波を透過させる技術
🏥臨床的意義
  • 後方エコー増強：嚢胞・膀胱など液体貯留部位の後方が明るくなる → 液体の存在を示す重要サイン
  • 音響陰影：骨・結石の後方でエコーが欠損 → 減衰・透過不良のサイン
  • プローブ選択の根拠：深部観察では低周波（透過↑）、浅部では高周波（分解能↑）
🔍関連概念・鑑別
後方エコー増強 vs 後方エコー減衰：前者は透過性↑（液体）で後方が明るい、後者は透過性↓（硬い腫瘤）で後方が暗い。この違いで内部性状を推定できる。
⚠️注意点
「後方エコー増強があれば嚢胞」と断言しない。まれに粘液腫瘍など均質な内部を持つ固形腫瘤でも生じることがある。
画像の質と血流計測
D4.10
解像度（分解能）
Resolution
▾
🔰初学者向け説明
カメラの「解像度」と同じ概念です。解像度が高いほど細かいものを見分けられます。超音波では「縦方向（ビームの進む方向）」「横方向（ビームに直交する方向）」「厚み方向」の3種類の分解能があり、それぞれの改善方法が異なります。
📖定義
2点を別個の点として識別できる最小距離。小さいほど分解能が高い（良い画質）。
①距離分解能（軸方向）：パルス幅で決まる
②方位分解能（側方向）：ビーム幅で決まる
③スライス方向分解能（厚み方向）：ビームの厚みで決まる
④コントラスト分解能：輝度差の識別能
⚙️メカニズム
  • 距離分解能：パルス幅が短いほど向上。高周波数 → 短パルス → 良距離分解能
  • 方位分解能：ビームが細いほど向上。フォーカス、電子リニアアレイで改善
  • スライス分解能：音響レンズによるビームの厚み方向の絞り込み
  • 多段フォーカス・ダイナミックフォーカスで広範囲の分解能を向上
  • オブリークスキャン：斜め入射で体表平行面（血管壁など）の分解能を改善
🏥臨床的意義
  • フォーカスを目標構造物の深さに合わせることが基本。FAST/POCUSでは関心領域に合わせる
  • 高周波プローブ（線形）vs 低周波（コンベックス）の選択は分解能-透過性のトレードオフ
  • ゲイン過多 → 輝度が均一化 → コントラスト分解能低下
🔍関連概念・鑑別
距離分解能 vs 方位分解能：距離は「音波の短さ」（周波数依存）、方位は「音波の細さ」（ビーム幅・フォーカス依存）。前者は周波数を上げることで、後者はフォーカス調整で主に改善する。
⚠️注意点
スライス幅アーチファクト：ビームの厚み内に存在する反射体が同一断面に表示されてしまう。嚢胞内に点状エコーが出現して内部充実性と誤認することがある（偽エコー）。
D4.11
エイリアシング
Aliasing
▾
🔰初学者向け説明
映画で扇風機の羽根が「逆回転」に見えることがあります。これはカメラの撮影速度（フレームレート）が羽根の速度に追いつけないために起こる錯覚です。超音波のエイリアシングも全く同じ仕組みで、血流が速すぎると計測装置が追いつけず、「実際とは逆向き」の血流として表示されてしまいます。
📖定義
パルスドプラ法において、血流速度がナイキスト限界（PRF/2）を超えた場合に発生する「折り返し現象」。実際の流れとは逆向き・逆色で表示されるアーチファクト。
⚙️メカニズム
  • PRF = パルス繰り返し周波数（1秒間の送受信回数）
  • ナイキスト限界 = PRF / 2。これを超えるドプラシフトは正しくサンプリングできない
  • パルスドプラ波形：基線を越えて上下が入れ替わって表示
  • カラードプラ：赤↔青が急激に反転表示
🏥臨床的意義
  • 対処①：PRF（速度レンジ）を上げる → より高速な血流を計測可能に
  • 対処②：ベースライン（基線）を移動 → 一方向の表示範囲を拡大
  • 対処③：連続波ドプラ（CWD）に切り替え → 原理的にエイリアシングが生じない
  • 対処④：低周波プローブに変更 → ドプラシフト量が減少
  • 狭窄部・弁逆流など高速血流の評価で頻発。カラードプラでモザイクパターンとして現れ、血流乱流のサインにもなる
🔍関連概念・鑑別
  • エイリアシング vs 乱流：乱流はカラードプラでモザイク様表示（多方向の色混在）、エイリアシングは色が鏡面的に「反転」する点が異なる
  • パルスドプラ vs 連続波ドプラ：前者はエイリアシングあり（深度弁別能あり）、後者はなし（深度弁別なし）
⚠️注意点
PRFを上げすぎると最大深度が浅くなる（深い部位から送受信する時間が確保できなくなる）。深部の高速血流計測では連続波ドプラが安全策。
💡 Claude 補足
  • 音速の固定値について：装置は生体内音速を1,540 m/s（または1,540 m/s近似）として距離計算を行う。実際には脂肪（約1,450 m/s）や骨（約3,500 m/s）で大きく異なり、これが「音速アーチファクト」（深さの誤計算）の原因になる。NotebookLMの資料には記載がないため補足。
  • 減衰係数の具体値：軟部組織での減衰係数は概ね 0.5 dB/cm/MHz。5MHz・10cmでは 5×10×0.5 = 25 dB 減衰（往復で50 dB）。試験では「周波数×深さ（cm）×0.5」の計算が問われることがある。
  • エイリアシングの対処に「角度修正」は含まれない：ドプラ角度を小さくすると検出速度が見かけ上下がりエイリアシングが悪化することもある。対処の優先順位は①PRF↑、②ベースライン移動、③CWD切替。
  • 整合層（matching layer）：D4.9で言及したプローブ設計の重要技術。λ/4の厚みの材料を振動子前面に設け、音響インピーダンスを段階的に変化させることで透過効率を最大化する。臨床現場では意識しないが原理として知っておくと良い。
