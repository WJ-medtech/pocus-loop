D4.12
Bモード（輝度モード）
B-mode / Brightness Mode
▾
🔰初学者向け説明

白黒写真を撮るカメラに例えると、超音波を体に当てて「跳ね返ってきた音の強さ」を明るさに変換した"体の断面写真"です。

骨や石灰化のように音をよく反射するものは白く、水や液体のように音をほとんど透過するものは黒く映ります。POCUSの「見た目」のベースになるのがこのモードです。

📖定義

超音波パルスの反射強度（振幅）を輝度（Brightness）の強弱に変換し、2次元断層像として表示する描画法。超音波断層撮影の最も基本となるモード。反射が強いほど白く（高輝度）、弱いほど黒く（低輝度）表示される。

⚙️メカニズム
  • 超音波パルスを送信し、組織で反射して戻る時間から距離を算出（音速 ≈ 1,530〜1,540 m/s と仮定）
  • 1回の送受信で1ライン（ラスター）を取得
  • ビーム方向を少しずつずらしながら（スキャン）繰り返すことで2次元断面像を構築
  • 深部ほど信号が減衰するため、STC/TGC（深度補正回路）により深さ全体にわたって均一な輝度になるよう補正
  • 微弱な信号の差を表示するため対数アンプを使用
🏥臨床的意義

臓器の形態・サイズ・内部エコーパターン・境界の評価に使用。

  • 心嚢液・腹腔内遊離液・膀胱尿貯留・胸水など、液体貯留の確認は「無エコー（黒）の領域」として直感的に識別
  • 全てのPOCUS評価の出発点となるモード
💡 液体 → 黒（無エコー）／骨・石灰化 → 白（高エコー）／軟部組織 → 灰色（中等度エコー）
🔍関連概念・鑑別
  • Aモード（振幅モード）との違い：Aモードは反射強度を「グラフの高さ」で表す1次元表示。Bモードはそれを「輝点の明るさ」で2次元画像化したもの
  • Mモードとの関係：Bモードの1ラインを時間軸に展開したもの
⚠️注意点
  • 高エコー（白）≠ 必ず石灰化。脂肪・線維組織も白く映る
  • TGCの調整不足で深部が暗くなり病変を見逃すことがある
  • 音速 1,530 m/s の仮定は近似値。気体・骨では大きく異なる
D4.13
Mモード（モーションモード）
M-mode / Motion Mode
▾
🔰初学者向け説明

心臓の弁や壁が"ピコピコ動く様子"を、横向きのグラフとして記録するモードです。横軸が時間、縦軸が深さ。バウンシングボールの軌跡を横から撮影したようなイメージで、わずかな動きの量・速さ・タイミングを精密に測れます。

📖定義

同一走査線上で超音波の送受信を高速に繰り返し、その線上の輝度変化を時間軸（横軸）に沿って表示するモード（Motion mode）。縦軸は深さ（距離）、横軸は時間を示す。

⚙️メカニズム
  • プローブを動かさずに同一方向へ繰り返しパルスを送信
  • 各タイミングで反射してきた輝度データを横に並べることで「位置の時間変化」を可視化
  • 通常はBモードと同時に表示（B/Mモード）し、Bモード上でサンプル線を指定してMモード波形を取得
🏥臨床的意義
  • 心臓：僧帽弁・大動脈弁の開閉タイミング・振幅の計測、左室壁厚・内径の計測（EF推定）
  • IVC（下大静脈）：呼吸による虚脱の度合いから循環血液量の評価。IVC径 ≤ 2.1 cm ＋ 呼吸性虚脱 ≥ 50% → 右房圧低下（≤ 5 mmHg）と推定
  • 時間分解能が高く、弁の動きや壁運動の「タイミング異常」を検出しやすい
🔍関連概念・鑑別
  • Bモードとの違い：Bモードは空間的な2次元断面。Mモードは1直線上の時間的変化
  • スペクトラルドプラとの違い：Mモードは「位置の変化」、スペクトラルドプラは「血流速度の変化」を時間軸で追う
⚠️注意点
  • ビームを目的の構造物に正確に当てないと誤った動態情報を得る
  • 壁運動の「タイミング」解釈には心電図同期が理想的（R波と動きの対応）
  • POCUSでは簡易的なEF評価に用いるが、定量的精度はTTEに劣る
D4.14
カラーフローマッピング（CFM）
Color Flow Mapping
▾
🔰初学者向け説明

白黒のBモード画像の上に「血液の流れ」を色で塗り重ねる機能です。

🔴 赤：プローブに向かってくる血流（BART: Red Toward）
🔵 青：プローブから遠ざかる血流（BART: Blue Away）
🟢 緑混入：乱れた血流（赤→黄・青→シアン）

逆流や狭窄を"一目で"見つけるための地図です。

📖定義

ドプラ効果を利用してリアルタイムに血流情報（方向・平均流速・パワー・分散）を取得し、Bモード像上にカラーで重畳表示する方式。Color Flow Mapping（CFM）。

⚙️メカニズム
  • 同一ラインに対して複数回（十数回）パルスを送受信（マルチゲート）
  • 自己相関法で各点の平均流速・方向・乱れを算出
  • ライン単位で繋ぎ合わせてカラーマップとして表示
  • 送受信回数が多い分、Bモード単体に比べてフレームレートが低下する
🏥臨床的意義
  • 弁逆流（AR・MR・TR）の部位・方向・程度の視覚的把握
  • 心内短絡（VSD・PDA）の検出
  • 腹部血管・門脈血流の評価
  • DVTスクリーニング：圧迫法と組み合わせて血流の有無を確認
🔍関連概念・鑑別
  • パワーモードとの違い：CFMは「方向＋速さ」を色分けするが角度依存性が高い。パワーモードは「存在」のみで方向不明だが角度に強く感度が高い
  • エイリアシング：高速血流でPRFの1/2（ナイキスト限界）を超えると色が反転（赤↔青）
  • モザイクパターン：乱流（狭窄・逆流）で多色混在表示
⚠️注意点
  • ビームと血流が直交（90°）すると血流が映らない（ドプラシフト=0）
  • カラーゲインを上げすぎるとノイズが血流として表示される
  • フレームレートとカラー域の広さはトレードオフ。カラーボックスは小さく保つ
D4.15
パワードプラ
Power Doppler / Power Flow Mapping
▾
🔰初学者向け説明

「血液が流れているかどうか」だけに特化した、超高感度センサーです。方向も速さもわからないけれど、ほんの少しでも流れがある場所を漏れなくオレンジ色で表示します。細い腫瘍内血管や非常にゆっくりした末梢血流を見つけるのが得意です。

📖定義

ドプラシフトした信号の反射強度（パワー）の積分値を輝度・色で表示するモード。血流の「存在量」を強調し、方向・速度情報は持たない。「パワーフローマッピング」とも呼ばれる。

⚙️メカニズム
  • 血球からの散乱信号のパワー（強度の積分）を検出
  • 十数回の加算平均処理でノイズを抑制しつつ微細信号を強調
  • 角度依存性が極めて低い（90°でも血球による散乱シグナルが得られる）
🏥臨床的意義
  • 腹部臓器（肝臓・腎臓・脾臓）実質の血流分布評価
  • 末梢血管、細小血管の走行描出
  • 炎症・腫瘍における血流増生の評価（新生血管は低流速）
  • DVTスクリーニングの補完（特に遠位部・細い血管）
🔍関連概念・鑑別
  • CFM（流速モード）との比較：
CFM：向きと速さ → 角度に弱い・エイリアシングあり・方向わかる
パワードプラ：存在 → 角度に強い・エイリアシングなし・方向わからない
⚠️注意点
  • 逆流の有無など「方向」が必要な評価には使えない
  • フラッシュアーティファクト：プローブや患者の動きを血流と誤検出しやすい
  • 過剰ゲインで組織への色づき（ブルーミング）が起きる
D4.16
スペクトラルドプラ
Spectral Doppler (PWD / CWD)
▾
🔰初学者向け説明

血流の「速度」を音楽の楽譜のように可視化する機能です。横軸が時間、縦軸が速度で、「この瞬間に◯cm/sで流れている血液がこれだけある」をリアルタイムで波形として描きます。速すぎる血流（狭窄）や逆流・異常な波形パターンを数値で定量評価できる精密検査ツールです。

📖定義

ドプラ信号を高速フーリエ変換（FFT）で周波数解析し、時間軸に沿って速度（または周波数偏移）の分布をグラフ状に表示する方法。パルスドプラ法（PWD）および連続波ドプラ法（CWD）の出力形式として用いられる。

⚙️メカニズム
  • FFT：複雑な反射波を周波数成分に分解し、各速度成分の強さをグラフ化
  • パルスドプラ（PWD）：特定深度（サンプルボリューム）のみを解析。距離分解能あり。PRFの1/2（ナイキスト限界）を超えるとエイリアシングが発生
  • 連続波ドプラ（CWD）：送受信を連続して行い、ビーム上の全深度を積算。エイリアシングなし・高速血流計測可能だが距離分解能なし
速度計算式：v = (Δf × c) ÷ (2 × f₀ × cosθ)
角度補正は必須（推奨 θ ≦ 60°）
🏥臨床的意義
  • 心臓弁膜症：大動脈弁狭窄のVmax・圧較差計算
  • 血管狭窄度の定量評価：PSV・EDV・RI・PI の算出
  • 門脈血流パターン（肝硬変・門脈圧亢進）
  • 産科：臍帯・中大脳動脈血流評価
  • 腎動脈狭窄スクリーニング
🔍関連概念・鑑別
  • PWD vs CWD：「場所の特定ができる/できない」「高速血流に弱い/強い」
  • CFMとの連携：CFMで逆流・狭窄の「場所」を同定 → PWD/CWDで「速度の定量」
  • Mモードとの違い：Mモードは「構造物の位置変化」、スペクトラルドプラは「血流速度の変化」を時間軸で追う
⚠️注意点
  • 角度補正を怠ると速度を大幅に過小評価（cosθが小さくなる）
  • PWDでエイリアシングが出たら → PRFを上げる・ベースラインをシフト・CWDへ切替
  • CWDは「ビーム上のどこの血流か不明」なため、複数構造が重なる場合に波形の帰属が困難

💡 Claude補足

① 音速の値について
資料では音速を「1,530 m/s」と記載していますが、一般的なPOCUS・超音波の教科書では生体軟部組織の平均音速は 1,540 m/s（または1,538 m/s）が標準値として使われます。試験・臨床文書では1,540 m/sを用いるのが通例です。
② BARTの原則について
CFMの「赤＝近づく・青＝遠ざかる」はBARTの原則（Blue Away, Red Toward）として記憶するのが一般的ですが、カラーマップは装置設定で反転可能です。常にプローブとの位置関係を確認してから解釈してください。
③ ナイキスト限界（Nyquist limit）の補足
エイリアシング発生条件「PRFの1/2を超えると」という記述はナイキスト‐シャノンの標本化定理に基づきます。この「PRFの1/2」をナイキスト限界と呼び、装置のカラースケール上限として表示されます。臨床ではエイリアシングが「狭窄の指標」として逆に有用なこともあります。
④ 組織ドプラ（TDI）について
スペクトラルドプラの応用として組織ドプラ（TDI: Tissue Doppler Imaging）があります。血流ではなく心筋の動きをドプラで計測するもので、e'速度の計測によるLV拡張能評価（E/e'比）に使われます。今後のチャプターで登場した際に関連づけて理解してください。
