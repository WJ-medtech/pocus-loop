定義を問う問題
Q1 初級 定義
超音波（ultrasound）の定義として正しいのはどれか。
正解 C

超音波の定義は「人間の可聴域を超える周波数の音波」です。可聴域の上限は約20,000Hz（20kHz）なので、20kHzを超えると超音波になります。診断用超音波はこれよりはるかに高い2〜20MHzを使用します。

  • A：「2MHz以上」は診断用超音波の周波数範囲だが、超音波の定義ではない
  • B：振幅は音の強さに関するパラメータで定義には無関係
  • D：波長ではなく周波数で定義する
Q2 初級 定義
音響インピーダンス（Z）の計算式として正しいのはどれか。
正解 B

Z = ρ × c（密度 × 音速）が音響インピーダンスの定義式です。単位はRayl（レイル）またはkg/m²/s。2つの組織のZ差が大きいほど境界での反射が強くなります。

  • A：c = f × λ は音速の式（インピーダンスではない）
  • C：Z = c/ρ ではなく ρ×c（順番が逆でも意味が異なる）
  • D：振幅は別概念
Q3 初級 定義
Bモード画像で「輝度（明るさ）」として表現されるのはどれか。
正解 C

Bモード（Brightness mode）では、受信したエコーの振幅（強さ）を輝度（白黒の明るさ）に変換して画像を形成します。振幅が大きい = 強い反射 = 白く明るく表示されます。

  • Aモード（Amplitude mode）では振幅を縦軸のグラフとして表示する
  • 周波数・波長は画像の質（分解能・深達性）を決めるが、輝度そのものではない
Q4 初級 定義
超音波の「減衰」を構成する3要素の組み合わせとして正しいのはどれか。
正解 B

減衰（Attenuation）＝ ①吸収（熱変換）＋ ②拡散（ビーム広がり）＋ ③散乱（微細構造による四散） の3要素の合計です。

  • 反射・透過は境界での現象（減衰の原因ではあるが、分類上「減衰」の構成要素とは区別される）
  • 屈折は進行方向の変化で、エネルギー損失とは別
  • 「吸散拡（きゅうさんかく）」と語呂合わせで覚えると良い
Q5 初級 定義
エイリアシング（aliasing）が発生するのはどのような状況か。
正解 C

エイリアシングはパルスドプラ法特有のアーチファクトで、ドプラ偏移が ナイキスト限界 = PRF/2 を超えた際に発生します。波形が基線を越えて折り返し、実際とは逆向きに表示されます。

  • 連続波ドプラ（CWD）は連続送受信のため原理的にエイリアシングが生じない
  • A：強反射 → 音響陰影（別現象）
  • D：斜め入射 → 屈折アーチファクト（別現象）
メカニズムを問う問題
Q6 初級 メカニズム
媒質中の音速が一定のとき、周波数と波長の関係として正しいのはどれか。
正解 B

c = f × λ（音速 = 周波数 × 波長）より、音速(c)が一定の場合、f と λ は反比例します。周波数が2倍になれば波長は1/2に短くなります。「高周波 = 短波長 = 高分解能」というセットで覚えましょう。

Q7 中級 メカニズム
超音波の距離分解能（軸方向分解能）を向上させるために最も有効な方法はどれか。
正解 C

距離分解能（軸方向）はパルス長（パルス幅）が短いほど向上します。高周波数 → 短波長 → 短パルス → 優れた距離分解能という関係があります。

  • A：フォーカスは方位分解能（側方向）に影響する
  • B：PRFは深度・エイリアシングに関係し、距離分解能には直接影響しない
  • D：TGCは画像の明るさ補正であり、分解能には影響しない
Q8 中級 メカニズム
超音波の減衰について正しい記述はどれか。
正解 C

減衰の特性：①高周波数ほど減衰係数が大きい（周波数依存減衰）、②振幅は伝搬距離に対して対数的（指数関数的）に減少する。

  • A：逆。高周波ほど減衰大（深部が見えにくくなる原因）
  • B：線形ではなく対数的（dBスケールでは線形）
  • D：減衰の3要素は「吸収・拡散・散乱」。反射は減衰の分類に含まない
Q9 中級 メカニズム
超音波の屈折について正しい記述はどれか。
正解 B

屈折は音速の異なる組織境界に斜めに入射した場合に生じます。進行方向が曲がる度合いはSnellの法則（sin θ₁/c₁ = sin θ₂/c₂）で決まります。

  • A：垂直入射では屈折しない（そのまま直進）
  • C：屈折は周波数ではなく音速差と入射角で決まる
  • D：スライス幅アーチファクトはビームの「厚み」が原因（屈折ではない）
Q10 中級 メカニズム
散乱（Scattering）が発生する主な条件はどれか。
正解 C

散乱は波長より小さい微細な不規則構造（赤血球、細胞レベルの組織構造など）に超音波が当たることで、あらゆる方向に反射波が生じる現象です。画像上の「スペックル（粒々模様）」の原因です。

  • A：大きな平滑面への垂直入射 → 鏡面反射（散乱ではない）
  • B：気体との境界 → ほぼ全反射（音響陰影の原因）
  • D：散乱は高周波（短波長）ほど、波長以下の構造に当たりやすいため強くなる傾向がある
Q11 中級 メカニズム
振動子の共振周波数(fz)の計算式として正しいのはどれか（c：音速、L：振動子の厚み）。
正解 B

振動子の共振周波数は fz = c / (2L) です。振動子の厚みLが大きいほど共振周波数は低くなります（厚い振動子 = 低周波プローブ）。この関係から、高周波プローブの振動子は薄く作られています。

Q12 中級 メカニズム
超音波の反射において、反射が最も強くなる条件はどれか。
正解 C

反射の強さは2つの条件で最大になります：①音響インピーダンス（Z）の差が大きいこと、②ビームが境界面に垂直（90°）に入射することです。例として骨や気体との境界（Z差が極大）に垂直入射するとほぼ全反射になります。

  • A：Z差がゼロ → 反射率 = 0（境界が見えない）
  • B：斜め入射 → 反射波がプローブに戻りにくくなり、見かけ上弱い反射となる
  • D：周波数は反射率（R）の直接の計算式に含まれない
臨床応用を問う問題
Q13 初級 臨床応用
腹部の深部臓器（肝臓・膵臓など）を観察する際に適切なプローブはどれか。
正解 C

深部観察には低周波数（2〜5MHz）のコンベックスプローブを選択します。低周波数ほど減衰が少なく深部まで届きます（深達性↑）。分解能は劣りますが、まず構造を把握するには十分です。高周波リニアプローブは表在（頸部・甲状腺・筋骨格など）専用です。

Q14 中級 臨床応用
TGC（Time Gain Compensation）の役割として正しいのはどれか。
正解 B

TGC（Time Gain Compensation）またはSTC（Sensitivity Time Control）は、受信エコーを時間（＝深さ）に応じて増幅するゲイン補正機能です。深部からの信号は減衰で弱くなっているため、その分だけゲインを上げて全体の明るさを均一にします。送信側（出力・周波数）とは別の機能です。

Q15 中級 臨床応用
パルスドプラ法でエイリアシングが生じた場合の対処として誤っているのはどれか。
正解 D（誤り）

TGCのゲイン増加は深部の明るさ補正のための機能であり、エイリアシングとは無関係です。エイリアシングはサンプリング速度（PRF）不足の問題です。正しい対処は：

  • ①PRFを上げる（ナイキスト限界を引き上げる）
  • ②ベースライン移動（一方向の表示範囲を拡大）
  • ③CWDに切替（エイリアシングが生じない）
  • ④送信周波数を下げる（ドプラシフト量が小さくなる）
Q16 中級 臨床応用
超音波の安全性に関して、「機械的作用（キャビテーション）」の指標として用いられるのはどれか。
正解 A

超音波の安全指標：MI（Mechanical Index）は音圧（振幅）に関連し、キャビテーション（気泡崩壊による機械的ダメージ）のリスクを示します。TI（Thermal Index）は吸収による熱発生（温度上昇）のリスクを示します。両者は独立した指標です。

  • 造影剤使用時はMI < 0.4を目安に管理
  • 胎児・眼球などではTI < 0.7を推奨
Q17 中級 臨床応用
「後方エコー増強（posterior acoustic enhancement）」が見られる病変はどれか。
正解 C

後方エコー増強：液体（水）は超音波の減衰が非常に少ないため、液体貯留部位（嚢胞・膀胱・胸水など）を通過した後方のエコーが相対的に明るくなります。これは「液体の存在を示す重要な所見」です。

  • A・D：石灰化・結石 → 音響陰影（後方が暗い）
  • B：硬い充実性腫瘤 → 後方エコー減衰（後方が暗い）
他概念との鑑別を問う問題
Q18 中級 鑑別
「音響陰影（acoustic shadow）」と「後方エコー減衰（posterior acoustic shadowing with attenuation）」の違いとして正しいのはどれか。
正解 C
  • 音響陰影：骨・結石・ガスなど強反射体の後方で超音波がほとんど透過できず、後方エコーが完全に欠損する
  • 後方エコー減衰：充実性腫瘤（線維腺腫など）内部での減衰が強く、後方が周囲より相対的に暗くなる。完全欠損ではない点が重要
  • 臨床上この区別は腫瘤の内部性状鑑別（液体か固形か、石灰化の有無）に重要
Q19 中級 鑑別
パルスドプラ法と連続波ドプラ法の違いとして正しいのはどれか。
正解 B

両者の本質的な違いは「深度弁別能 vs 速度計測上限」のトレードオフです：

  • パルスドプラ（PW）：特定深度のサンプルゲートで計測 = 深度弁別◎、しかしPRFにより速度上限あり = エイリアシングのリスク
  • 連続波ドプラ（CW）：ビーム全体の情報を取得 = 深度弁別✗、しかし連続送受信 = 速度上限なし
Q20 上級 鑑別
距離分解能・方位分解能・スライス方向分解能のそれぞれを改善する手段の組み合わせとして正しいのはどれか。
正解 C
  • 距離分解能（軸方向）：パルス長が短いほど向上 → 高周波数（短波長 → 短パルス）で改善
  • 方位分解能（側方向）：ビーム幅が細いほど向上 → フォーカス調整・電子ビーム収束で改善
  • スライス方向分解能（厚み方向）：ビームの厚みを絞る → 音響レンズ（固定焦点）で改善
  • PRFとTGCはそれぞれ速度計測範囲・明るさ補正であり分解能改善とは無関係
臨床シナリオ形式
Q21 初級 シナリオ
腹部エコーで右上腹部を観察しようとしたが、肋骨の後方（深部）が完全に黒く映り、肝臓の一部が見えない。この原因として最も適切なのはどれか。
正解 B

肋骨（骨）は音響インピーダンスが生体軟部組織と大きく異なり、超音波をほぼ全反射します。そのため肋骨後方には超音波が届かず、完全な黒い影（音響陰影）が生じます。肋間からプローブ位置を変えて観察するのが対処法です。

Q22 中級 シナリオ
脂肪肝が疑われる患者のエコー検査で、肝臓が全体的に明るく（高輝度）映り、深部に向かうにつれ急激に画像が暗くなった。この所見の物理的な原因として正しいのはどれか。
正解 B

脂肪肝の特徴的所見「ブライトリバー＋深部減衰」の原因：

  • 肝細胞内に蓄積した脂肪滴（波長より小さい微細構造）が散乱源となり、肝実質からの散乱波が増大 → 全体的な輝度上昇（ブライトリバー）
  • 散乱・吸収の増大により、深部への超音波エネルギーが急速に消費される → 深部の急激な減衰（暗化）
  • これが脂肪肝の半定量的評価にも利用される
Q23 中級 シナリオ
FAST（Focused Assessment with Sonography in Trauma）を施行中、腹腔内に黒い無エコー領域が認められた。その後方のエコーが明るくなっていた（後方エコー増強）。この所見が示すのはどれか。
正解 B

FASTで観察する腹腔内液体（腹水・血腹）は：①無エコー（黒）：液体は超音波をほとんど反射しない、②後方エコー増強：液体の透過性が高いため後方のエコーが明るくなる、という2つの特徴を示します。これが液体貯留の典型的なエコー所見です。石灰化・腸管ガスは音響陰影を伴います。

Q24 上級 シナリオ
腹部エコー中、上腸間膜動脈（SMA）が二重に見えるダブルイメージを認めた。この原因として最も考えられるのはどれか。
正解 B

SMAのダブルイメージは屈折アーチファクトの典型例です。腹直筋（音速が周囲と異なる）に超音波が斜めに入射すると屈折が生じ、その先の血管が曲がったビーム上に「もう一本」存在するかのように描出されます。対処はプローブ位置・角度を変えて確認することで、虚像は消失します。

  • 多重反射：強反射体間をビームが往復 → 等間隔の複数ライン
  • サイドローブ：メインビーム外の不要ビームによる虚像（通常は強反射体周囲）
Q25 上級 シナリオ
心エコーで大動脈弁狭窄症（AS）の最大血流速度を計測しようとしたが、パルスドプラ（PW）ではエイリアシングが解消できなかった。次の対応として最も適切なのはどれか。
正解 C

重症ASでは大動脈弁口を通過する血流速度が4〜5 m/sを超えることがあり、パルスドプラのPRFを最大にしてもエイリアシングが解消できない場合があります。この場合は連続波ドプラ（CWD）に切り替えます。CWDは原理的にエイリアシングが発生せず、高速血流の正確な計測が可能です（深度弁別能は失われますが、ASの評価では問題になりません）。

Q26 上級 シナリオ
エコーゼリーを使用せずにプローブを皮膚に当てたところ、全く画像が得られなかった。この物理的理由として正しいのはどれか。
正解 B

空気の音響インピーダンスは生体軟部組織に比べて極めて小さく、反射率 ≒ -1.00（ほぼ100%反射）となります。ゼリーにはこの空気層を除去し、プローブ-皮膚間のインピーダンスマッチングを行う役割があります。ゼリーなしでは超音波が体内に届かず、画像が形成されません。

Q27 上級 シナリオ
エコーで嚢胞が疑われる病変内部に点状エコーが見えた。スライス幅アーチファクトとの鑑別に有用なアプローチはどれか。
正解 C

スライス幅アーチファクトはビームの「厚み」内に隣接する高輝度構造（血管壁など）が同一断面に描出されて、嚢胞内部に点状エコーが出現する現象です。プローブの位置・角度を変えると、ビーム幅内に取り込まれる構造が変わるため、偽エコーは変化・消失します。一方、真の内部エコー（内容物）は消失しません。この鑑別が重要です。

Q28 初級 シナリオ
妊婦の腹部エコーを施行する際、安全性の観点から特に注意すべき指標の組み合わせはどれか。
正解 C

超音波の安全指標：TI（Thermal Index）は吸収による熱発生リスク（胎児神経系に影響の可能性）、MI（Mechanical Index）はキャビテーションリスクを示します。いずれも画面上に表示されるため、確認しながら検査を行います。特に妊娠初期（第一三半期）では低い値を維持することが推奨されます（ALARAの原則）。

Q29 上級 シナリオ
5MHz・深さ8cmで観察中、深部エコーの振幅はどの程度になるか（軟部組織の減衰係数を0.5 dB/cm/MHzとし、往復の伝搬距離を考慮する）。
正解 D

計算：

  • 片道減衰 = 0.5 dB/cm/MHz × 5 MHz × 8 cm = 20 dB
  • 往復（送信8cm＋受信8cm）= 20 × 2 = 40 dB
  • 40 dBの減衰 = 振幅比で1/100（20 dBで1/10、40 dBで1/10 × 1/10 = 1/100）
  • 参考資料の「5MHz・8cmで1/10になる」は片道の計算を指している
  • 実際の装置はTGCでこの減衰を補正している
