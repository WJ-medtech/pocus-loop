Q1初級
Bモードにおいて、超音波画像上で「黒く（無エコー）」映る組織はどれか。
解説を見る ▾
✓ 正解：C — 膀胱内の尿

Bモードでは超音波の反射強度が輝度（明るさ）に変換される。液体（尿・胸水・腹腔内液など）は超音波をほぼ透過させるため反射がなく、無エコー（黒）として表示される。胆石・骨は強反射体で高輝度（白）を示し、後方音響陰影を生じる。肝実質は中等度エコーで灰色に映る。

Q2初級
Bモードの「B」が意味するものはどれか。
解説を見る ▾
✓ 正解：B — Brightness（輝度）

BモードのBはBrightness（輝度・明るさ）の略。超音波の反射振幅を輝点の明るさで表現することからこの名前がついた。これに対しAモードはAmplitude（振幅）、MモードはMotion（動き）を意味する。

Q3初級
Mモードで縦軸・横軸が示すものの組み合わせとして正しいのはどれか。
解説を見る ▾
✓ 正解：B — 縦軸：深さ　横軸：時間

Mモードは同一走査線上の輝度変化を時間軸に沿って記録する。縦軸が深さ（距離）、横軸が時間。これにより心臓弁の開閉運動や壁の動きの「振幅」「速度」「タイミング」を精密に評価できる。なお、スペクトラルドプラも横軸が時間だが、縦軸は速度（または周波数偏移）である点で異なる。

Q4初級
カラーフローマッピング（CFM）でプローブに向かって流れる血流は通常何色で表示されるか。
解説を見る ▾
✓ 正解：C — 赤

BARTの原則：Blue Away（遠ざかる→青）、Red Toward（近づく→赤）。ただし装置設定で反転できるため、スケールバーを確認することが重要。乱れた血流（分散）には緑が加わり、赤系は黄色、青系はシアンに変化する（モザイクパターン）。

Q5初級
パワードプラの最大の特徴はどれか。
解説を見る ▾
✓ 正解：C — 低流速・細血管の検出感度が高い

パワードプラはドプラ信号のパワー（反射強度）を表示するため、低流速・細血管でも感度よく描出できる。方向情報はなく（A誤り）、定量評価はスペクトラルドプラの役割（B誤り）、角度依存性は極めて少ない（D誤り）。

Q6中級
Bモードで深部が暗く見える場合、まず調整すべき機能はどれか。
解説を見る ▾
✓ 正解：C — TGC（Time Gain Compensation）

生体内を伝播する超音波は深部ほど減衰する。TGC（STC：Sensitivity Time Control）は深度に応じて受信ゲインを補正する機能で、深部が暗い場合はTGCの深部側スライダーを上げることで均一な輝度が得られる。カラーゲインはカラードプラ専用（A誤り）、PRFはドプラのレンジ設定（B誤り）、サンプルボリュームはPWDのゲート位置（D誤り）。

Q7中級
CFMにおけるエイリアシングが発生する条件として正しいのはどれか。
解説を見る ▾
✓ 正解：B — 血流速度がPRFの1/2を超えたとき

ナイキスト‐シャノンの標本化定理により、PRFの1/2（ナイキスト限界）を超える血流速度は正しく計測できず、色が反対側に折り返して表示される（エイリアシング）。対策はPRFを上げる、ベースラインをシフトする、またはCWDに切り替えることである。

Q8中級
パワードプラとCFM（流速モード）の比較として正しいのはどれか。
解説を見る ▾
✓ 正解：C — パワードプラはCFMより低流速血流の検出に優れる

パワードプラは信号パワーの加算平均処理により低流速血流の検出感度が高い。角度依存性はCFMより低く（A誤り）、方向情報はない（B誤り）。CFMは速度に基づくためエイリアシングが発生し、パワードプラではエイリアシングが生じない（D誤り）。

Q9中級
パルスドプラ（PWD）と連続波ドプラ（CWD）の違いとして正しいのはどれか。
解説を見る ▾
✓ 正解：C — PWDはサンプルボリュームで深度を指定できるが、CWDはできない

PWDは特定深度（サンプルボリューム）からの信号のみを解析するため距離分解能がある。CWDは超音波を連続送受信するためビーム上の全深度が加算され、距離分解能はないがエイリアシングなしで高速血流（弁狭窄など）を計測できる。A・B・Dはいずれも逆の記述。

Q10中級
スペクトラルドプラで正確な血流速度を計測するために推奨されるビームと血流のなす角度はどれか。
解説を見る ▾
✓ 正解：C — 60°以内

ドプラ速度式 v = Δf·c ÷ (2·f₀·cosθ) において、θが90°に近づくとcosθ→0となり速度が過小評価される。60°を超えると誤差が急増するため、60°以内の入射角が推奨されている。理想は45°以内だが、臨床的に許容される上限が60°とされる。

Q11中級
CFMでカラーボックス（関心領域）を小さく保つべき主な理由はどれか。
解説を見る ▾
✓ 正解：B — フレームレートを維持するため

CFMは同一ラインに対して十数回の送受信を行うため、カラーボックスが大きいほど処理すべきライン数が増え、フレームレート（画像更新速度）が低下する。特に心臓のような高速運動臓器では動きを追いきれなくなる。カラーボックスをできる限り小さく保つことがフレームレート維持の基本的な対策である。

Q12上級
CFMで「モザイクパターン」が観察される臨床的意義として最も適切なのはどれか。
解説を見る ▾
✓ 正解：C — 高速で乱れた血流（狭窄・逆流など）が存在することを示す

モザイクパターンは赤・青・緑・黄・シアンなどの色が混在した像で、速く乱れた血流（高速＋高分散）を反映する。弁逆流、弁狭窄、心内短絡（VSD）などで典型的に出現し、逆流ジェットの位置・方向・広がりの視覚的指標となる。遅い血流では単色のカラーで表示される（A誤り）。

Q13上級
大動脈弁狭窄症が疑われる患者で、最大血流速度を正確に計測したい。胸骨上窩アプローチで血流速度が5 m/s以上が予想される場合、最も適切なドプラモードはどれか。
解説を見る ▾
✓ 正解：D — 連続波ドプラ（CWD）

大動脈弁狭窄の最大血流速度は4〜6 m/s以上になることがあり、これはPWDのナイキスト限界（通常1〜2 m/s程度）を大きく超える。このような高速血流の定量計測にはCWDが適している。CWDはエイリアシングなしで高速血流を計測できる。実践的なワークフローとしては、CFMで弁口部を確認し角度を合わせてからCWDを施行する。

Q14上級
IVC（下大静脈）の呼吸性虚脱率（Caval Index）を評価するために最も適したモードはどれか。
解説を見る ▾
✓ 正解：B — Mモード

IVCの呼吸性虚脱率（caval index）はIVCの最大径と最小径から算出する。Mモードを肝静脈合流部直下のIVCに当てると、呼吸に同期した径の変化が精密に記録・計測できる。IVC径2.1cm以下＋50%以上の虚脱で右房圧低下（≤5 mmHg）と推定される。CFMは血流の方向確認に使うが、径の計測には不向き。

Q15上級
パワードプラで「フラッシュアーティファクト」が発生する主な原因はどれか。
解説を見る ▾
✓ 正解：B — プローブや患者の動きによる組織の動きを血流と誤検出した

フラッシュアーティファクト（フラッシュノイズ）はパワードプラに特有の問題で、患者やプローブの急激な動きによって組織自体が動き、ドプラシフトが発生することで、あたかも広範囲に血流があるように表示される現象。ナイキスト限界超過はエイリアシング（A誤り）、低ゲインは信号欠損（C誤り）の原因。

Q16上級
スペクトラルドプラ波形でRI（抵抗指数）を計算する正しい式はどれか。
解説を見る ▾
✓ 正解：A — (PSV − EDV) ÷ PSV

RI（Resistive Index）= (PSV − EDV) ÷ PSV（PSV：収縮期最高速度、EDV：拡張末期速度）。RIは0〜1の値をとり、高値ほど末梢血管抵抗が高いことを意味する（例：腎動脈RI > 0.7で腎血管障害疑い）。B式はS/D比（Systolic/Diastolic ratio）、C式は平均流速の近似式。
